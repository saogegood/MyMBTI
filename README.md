# MBTI测试页面

## 项目介绍

本项目实现了MBTI（Myers-Briggs Type Indicator）测试，包含93道题目（标准版），根据四维度（8种心理类型）组成的16性格测试。测试结果将是16性格中的一种，有助于认清自我，找到未来的自己向期望的方向发展。
